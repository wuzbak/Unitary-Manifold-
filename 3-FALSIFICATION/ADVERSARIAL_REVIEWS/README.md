## ⚑ The Two Types of Document in This Folder

### Type 1 — Adversarial / External Reviews

Documents produced by AI reviewers operating under instructions to challenge,
stress-test, or disprove the framework. These are **not the repository's own
view** of the work — they are deliberate adversarial challenges intended to
find every valid objection.

### Type 2 — Repository Responses and Conclusions

Documents in `RESPONSES/` represent the **repository's own understanding**:
how each challenge was resolved, what the framework's current honest status is,
and what the formal scientific conclusion is at each version milestone. These are
not external reviews — they are the authors' considered response.

**The distinction matters:** a finding in a Type 1 document that is not
reproduced in a Type 2 document has been evaluated and either resolved, absorbed
into `FALLIBILITY.md`, or addressed by a new Pillar.
