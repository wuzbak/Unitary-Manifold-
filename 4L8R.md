## AXIOM Architecture

### Five-Agent System

| Agent | Role |
|-------|------|
| `researcher` | Web search, public records retrieval, document acquisition |
| `fact_checker` | Cross-reference claims, detect contradictions, assign confidence |
| `legal_auditor` | Statute awareness, whistleblower protections, libel risk |
| `editor` | Brief synthesis, narrative structure, editorial gate |
| `orchestrator` | Coordinates agents, manages investigation pipeline, parallel threads |

### Six-Stage Investigation Pipeline

1. **Lead intake** — parse and structure the incoming lead
2. **Entity mapping** — identify all named persons, orgs, shell companies, agencies
3. **Research** — web search + public records (FOIA, PACER, SEC, GovInfo, OpenSanctions)
4. **Cross-reference** — deduplicate, score confidence, flag contradictions
5. **Legal audit** — flag legally sensitive claims, whistleblower implications
6. **Brief generation** — structured report with confidence scores and source tiers

### Single-Agent Mode (Truth Threshold Loop)

After each tool-calling round the agent self-assesses confidence (0.0–1.0). Research
continues until confidence reaches `TRUTH_THRESHOLD`. Max iterations: `JOURNALIST_MAX_ITERATIONS`.

### Multi-Agent Mode (`JOURNALIST_USE_MULTI_AGENT=true`)

Full `JournalismOrchestrator` with parallel research threads across all five specialized agents.

## Source Trust Tier System

| Tier | Score Weight | Examples |
|------|-------------|---------|
| 1 | 1.00 (CONFIRMED) | Primary gov docs, SEC filings, court records, FOIA responses |
| 2 | 0.85 (CONFIRMED) | Official agency reports, peer-reviewed sources |
| 3 | 0.65 (CORROBORATED) | Established news organizations, expert testimony |
| 4 | 0.40 (ALLEGED) | Secondary sources, unverified expert statements |
| 5 | 0.15 (UNVERIFIED) | Anonymous tips, unverified social media |

**Confidence labels:**
- ≥ 0.85 → `CONFIRMED`
- ≥ 0.65 → `CORROBORATED`
- ≥ 0.40 → `ALLEGED`
- < 0.40 → `UNVERIFIED`
