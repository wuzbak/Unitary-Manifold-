# CRITICAL FAILURE — Empirical No-Go Thresholds (Pillar 256)

Status: **ACTIVE**

This file defines three measurable results that are **forbidden** by the current framework.
If any one of these is confirmed at threshold, the corresponding sector is considered falsified.

## FST-1 — Sterile-Neutrino eV Discovery (Lepton Nullifier)

- **Forbidden signal:** A confirmed sterile neutrino mass eigenstate in the window **0.5–5.0 eV** with active-sterile mixing **sin²(2θ) > 0.02**.
- **Why this is a hard failure:** The current lepton-sector closure does not allow an eV sterile branch in this coupling range.
- **Measurement channel:** CERN Open Data neutrino archives (NOMAD channel) and cross-checkes with modern sterile-neutrino searches.
- **Data anchors:**
  - https://opendata.cern.ch
  - https://opendata.cern.ch/search?experiment=NOMAD

## FST-2 — QCD Running Chain Break in CERN Open Data

- **Forbidden signal:** Precision data establish **α_s(1 TeV) < 0.079** or **α_s(1 TeV) > 0.093** after validated systematics.
- **Why this is a hard failure:** The geometric running chain anchored at M_Z and propagated through the manifold cannot absorb this break without invalidating the QCD closure lane.
- **Measurement channel:** CMS public jet / event-shape analyses served through CERN Open Data.
- **Data anchors:**
  - https://opendata.cern.ch/search?experiment=CMS
  - https://cms-results.web.cern.ch/cms-results/public-results/publications/SMP/

## FST-3 — 0νββ Effective Mass Outside the Allowed Window

- **Forbidden signal:** A robust neutrinoless double-beta effective mass measurement outside **55–200 meV** (e.g., <30 meV or >250 meV at ≥3σ).
- **Why this is a hard failure:** The Majorana branch in the current geometric construction is constrained to the KK-scale window; values outside this interval break the neutrino-mass closure assumptions.
- **Measurement channel:** KamLAND-Zen / LEGEND / nEXO class measurements, interpreted with neutrino-sector consistency checks.
- **Data anchors:**
  - https://www.awa.tohoku.ac.jp/kamlandzen/
  - https://legend-exp.org
  - https://opendata.cern.ch

## Decision Rule

A theory that can explain every possible result is non-falsifiable. This file defines explicit **NO** thresholds.
Any threshold crossing requires immediate status update to a falsified state for the affected sector.
