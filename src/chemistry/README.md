modeling framework for chemistry.  Every quantity in this package is defined in
terms of φ-fields, and the equations follow internally from those definitions.

## What passing tests mean here

> A passing test confirms that the code faithfully implements the stated
> definition (e.g., "disease = φ-deviation from homeostasis fixed point").
>
> It does **NOT** confirm that chemistry phenomena are actually governed by
> 5D Kaluza–Klein geometry.

The definition could be wrong.  The test cannot see that.

## What this is useful for

This module is best understood as: *"if you model chemistry using this
mathematical structure, here is what follows."*  That may yield useful
frameworks for thinking about complex systems in chemistry.

## What this is NOT

This is not a physics proof about chemistry.  There is no single experiment
that would confirm or refute the claim that chemistry dynamics are fundamentally
governed by the Walker–Pearson field equations.
