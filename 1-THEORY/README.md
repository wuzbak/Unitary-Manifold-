## Epistemic Standards Applied Here

Claims in this folder are labelled using the following status vocabulary,
defined precisely in `DERIVATION_STATUS.md`:

| Status | Meaning |
|--------|---------|
| **PROVED** | Follows from the axioms by pure mathematics, no observational data |
| **DERIVED** | Follows from the axioms and the 5D metric ansatz by calculation |
| **PARTIALLY-DERIVED** | Most steps are derivations; one or more steps use physical arguments not yet geometrically proved |
| **PHYSICALLY-MOTIVATED** | The physics argument is compelling and self-consistent but the geometric proof is incomplete |
| **OBSERVATIONALLY-SELECTED** | The value is chosen from a discrete set constrained by theory; the data picks one member |
| **IDENTIFIED** | An object in the theory is associated with a known physical entity; this association is plausible but not derived |
| **OPEN** | Explicitly unresolved — documented to avoid overclaiming |

Nothing here is hidden; nothing is overclaimed.
