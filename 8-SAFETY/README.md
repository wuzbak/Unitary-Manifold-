# 8-SAFETY — Dual-Use Safety and Security

This folder contains safety protocols, radiological safety analysis, and
dual-use review documentation for the Unitary Manifold framework.

The `SAFETY/` subfolder contains:

| File | Purpose |
|------|---------|
| `README.md` | Safety overview and policy statement |
| `RADIOLOGICAL_SAFETY.md` | Cold fusion / LENR radiological risk analysis |
| `PROOF_OF_UNIQUENESS.md` | Uniqueness proof for the safe operating regime |
| `admissibility_checker.py` | COP admissibility checker for proposed experimental parameters |
| `thermal_runaway_mitigation.py` | Thermal runaway mitigation protocols |
| `unitarity_sentinel.py` | Unitarity sentinel: anomaly detection for field evolution |

See also `DUAL_USE_NOTICE.md` and `PENTAD_PRODUCT_NOTICE.md` at the repository root
for the dual-use stub policy and product protection policy.
