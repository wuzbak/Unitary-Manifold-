   208-pillar physics framework, the Unitary Pentad governance layer, or the
   adjacent non-hardgate research tracks (Pillars 218–243).

3. **No scientific claim** was ever attached to the Unitary OS in a hardgate
   context. It was purely a software engineering side project.
