## 6) Hard Gate Criteria (Promotion Rules)

A status promotion is valid only if all conditions pass:
1. **Mathematical traceability:** derivation reaches 5D action/topological roots.
2. **Numerical closure:** <5% residual for closure-class targets (or theorem-exact where claimed).
3. **Epistemic integrity:** no category mixing; caveats explicit in canonical ledgers.
4. **Validation integrity:** tests pass; no regression.
5. **Reproducibility:** callable module + tests + audit note exists for each promoted claim.
